# Security Policy

## Supported Versions

Agent Trigger Kit is currently in the `0.x` stage. Security fixes are applied
to the latest version on `main`.

## Reporting A Vulnerability

Please do not open a public issue for a suspected vulnerability.

Report security concerns through GitHub's private vulnerability reporting for
this repository when available. If that is unavailable, contact `CCC0509`
through GitHub.

Please include:

- A concise description of the issue.
- Steps to reproduce or a proof of concept.
- The affected files, commands, or generated outputs.
- Any known impact or mitigation.

We will acknowledge valid reports as soon as possible and coordinate a fix
before public disclosure.
